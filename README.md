# mobile-dev-journey

A documented transition into mobile development — Flutter first, then React Native, then native iOS/Android.

I'm coming from a Python background with no prior mobile experience. This repo tracks what I build and learn at each phase, in sequence.

---

## Roadmap

| Phase | Focus | Timeline |
|-------|-------|----------|
| 1 | Dart fundamentals | Weeks 1–3 |
| 2 | Flutter fundamentals | Weeks 4–10 |
| 3 | Firebase + REST APIs | Weeks 11–18 |
| 4 | Portfolio + job hunting | Weeks 19–30 |
| 5 | React Native | Months 8–14 |
| 6 | Native Swift or Kotlin | Months 15–24 |

**Current phase: 1 — Dart fundamentals**
